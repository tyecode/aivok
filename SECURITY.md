# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please open an issue with the label `security`. We will respond as quickly as possible.

For sensitive security issues, you may also contact the maintainer directly.

**Please do not report security vulnerabilities through public GitHub issues.**
